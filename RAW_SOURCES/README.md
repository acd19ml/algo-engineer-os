# RAW_SOURCES

原始资料层。论文、文档、课件、网页等第一手材料的入口。

## 这层和 INBOX 的区别

- **INBOX**：临时随手丢的所有东西（包括 dialogue_logs、ppts、cheat sheet）
- **RAW_SOURCES**：经过 triage 沉淀下来的、值得长期保留作为引用的原始材料

并不是所有 INBOX 内容都会进 RAW_SOURCES。课件、tutorial 等学习辅助通常留在 INBOX 不动；只有"长期会被多个 KNOWLEDGE 节点 / PROBLEMS 页引用的论文 / 标准文档"才升到这里。

## 子目录约定

按需建。常见类型：

- `papers/` — 学术论文
- `docs/` — 官方文档（如某框架文档抓取）
- `talks/` — 演讲、视频笔记

## 引用方向

KNOWLEDGE / PROBLEMS / PROJECTS 可以引用这里的条目。RAW_SOURCES 不向上引用。
