## 部门介绍

集团 IT（质量与流程 IT 部）是华为公司推动自身数智化转型的核心部门。业务应用与 IT 装备服务部作为其重要组成部分，负责打造企业级大规模 SaaS 产品，包括 MetaERP、MetaCRM、MetaIT、MetaSCM、Welink 等，为业务和用户提供数智化先进生产力装备。

部门支撑华为在产品研发、财经、营销销售、服务交付、供应链、采购、生产制造、人力资源、行政办公等全业务场景的智能化转型，目标是持续提升作业效率与用户体验，助力业务高效运营并增强客户满意度。

## AI 转型方向

当前企业应用走向 Agent，企业 SaaS 架构从元数据多租向 AI Native 演进，软件开发过程从人写代码演进到人指挥 Agent 写代码，包括 Code Agent、AI 辅助编码、AI 辅助设计、AI 辅助测试、AI 应用安全等。

部门聚焦以下场景与技术落地：

1. 构建 AI 应用核心工程能力 Harness Engineering，包括 AI 模型定制微调、企业知识 Agentic RAG、2B Agent 设计模式、跨领域 Agent 协同与集成、企业应用 Skills、AI 企业级安全隐私、企业级 Memory 机制、MCP & Agent & Skills 市场。
2. 打造效率倍增的垂直领域 2B Agent，包括员工助手 WeClaw Agent，研发 Agent、财经 Agent、供应链 Agent、采购 Agent、营销销售 Agent、服务交付 Agent、配置报价 Agent、行政办公 Agent、审计 Agent、人力资源 Agent 等。

## 岗位及方向

1. AI 应用工程师：AI 技术应用 / AI 系统软件
2. AI 模型工程师：Agent 技术 / 语言大模型 / 多模态大模型 / AI 算法 / 大模型数据技术 / 后训练与强化学习
3. AI 安全与隐私工程师：AI 应用安全

## 岗位职责与要求

1. 负责华为 SaaS 应用产品 MetaERP、MetaCRM、Welink 等智能化演进，在研发、财经、供应、采购、营销、销售、协同办公等 AI 转型落地场景中开发各类大模型 Agent。
2. 调用各种大模型服务，参与模型构建、训练、微调、部署及技术创新，确保 AI Agent 应用落地，解决具体业务问题并提升系统性能、稳定性与安全性。
3. 编程基础扎实，掌握 Python / Java / C++ 等至少一种编程语言。
4. 对数据敏感，具备 Agent、RAG、AI 算法、数据、模型（大 / 小模型）、Skills 等研究、使用、开发、设计、测试、部署、训练、推理、微调、安全、工程化、应用落地等相关经验之一或多项。
5. 求真务实、快速学习，愿长期致力于 B 端 AI 应用、Agent 和 AI 工程化落地方向，解决具体问题并做价值变现。
