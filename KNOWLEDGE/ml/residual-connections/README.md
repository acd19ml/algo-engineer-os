# Residual Connections

## 它要解决什么问题

深层网络已经有了初始化和归一化两道工程补丁来稳定训练，但**特别深的网络**（几十、上百层）仍然很难训。

残差连接是第三道补丁，但它的修补方式和前两个**性质不同**：初始化和归一化都在改"连乘里每一项的量级"，残差连接改的是 "**有没有一项一定是 1**"——它给信号和梯度开了一条**完全不经过非线性变换**的旁路。

## 朴素直觉为什么不够：普通块要学的是什么

普通网络里，一个块通常要直接学习一个映射：

```
y = H(x)
```

也就是输入 `x` 经过一堆变换后，**直接输出目标映射** `H(x)`。

如果这个块其实只需要"稍微改一下输入"——比如目标是 `H(x) ≈ x` 加一个小修正——那让网络直接学整个 `H(x)` 反而**别扭**：它必须学会 "**复制 x，并在此基础上加上微调**"。**复制**这件事看似简单，但用一个非线性变换组合精确地复制输入是有难度的。

## 残差块的形式：让网络学修正量而不是完整映射

残差连接把输出写成：

```
y = F(x) + x
```

- `x` 通过 **identity 分支**直接保留下来
- `F(x)` 是网络块学到的**修正量**
- 输出 = "原输入 + 修正"

这等价于让网络学：

```
F(x) = H(x) - x
```

也就是 **目标输出和原输入之间的差值**。

## 反事实：表达等价 ≠ 训练等价

聪明的反应是："`F(x) + x = H(x)`，从代数上看两种形式可以表达同一个最终映射，残差连接看起来什么也没改？"

代数上这个反应是对的，**但问题不是"最终能不能表达"，而是"训练过程中网络更容易学到哪一种东西"**。

举个最小例子。假设某一块**最理想的行为是 `H(x) = x`**（也就是这个块其实不需要改任何东西）：

- **普通网络** 必须学会 "**输入什么，就输出什么**"——在一个非线性变换组合里精确复制 x，**这并不容易**
- **残差网络** 只需要学 `F(x) = 0`，因为 `y = 0 + x = x`——**让一个网络块输出接近 0 通常比让它精确复制输入容易得多**

更一般地，如果目标 `H(x)` 离 `x` 不远（比如只需要"复制加微调"），那 `F(x) = H(x) - x` 是一个**较小的修正量**——较小的修正量通常更容易学。

**残差连接没有改变模型能表达什么，但改变了"网络要学的目标函数的形态"**——这才是关键。

## 为什么对梯度也有好处：来自两条依赖的梯度贡献相加

`y = F(x) + x` 在反向传播时还有第二个好处。看 `y` 对 `x` 的偏导：

```
dy/dx = dF(x)/dx + 1
```

这个 **`+1` 来自 identity 分支的直接依赖**。

注意一个常见误解：**这不是"梯度选择走哪条路"**——`F(x)` 这条路径仍然要计算、仍然参与反向传播。准确说法是：**`y` 对 `x` 有两种依赖（一种通过 `F(x)` 间接、一种通过 `+x` 直接），反向传播时这两部分梯度贡献相加**。

为什么这能缓解梯度消失？看深层网络的链式法则连乘：每经过一层乘上一个局部导数。如果 `dF(x)/dx` 这一项因为复杂网络变得很小（趋近 0），普通网络这一层贡献就接近 0。但残差块还有：

```
+1
```

**这个 1 直接保证这一层至少有一个不会消失的梯度贡献**。

更严谨说：`+1` 不保证所有情况下梯度绝对不消失，但它**显著降低了梯度被复杂变换链条彻底缩没的风险**——尤其是层数特别多的时候。

## 直觉收尾

残差连接的两个收益是相关但不同的：

- **优化难度**：网络要学的是修正量 `F(x) = H(x) - x` 而不是完整映射 `H(x)`。当目标接近恒等时，修正量接近 0，比"复制 + 微调" 更容易学。
- **梯度回传**：`dy/dx` 多出一项 `+1`，让反传不必完全依赖复杂的 `F(x)` 路径。

两者放在一起，让 50、100 层的网络**变得可训练**——这是 ResNet 之前不能做到的。

## Open Questions

- **Pre-norm 和 post-norm 残差块**（normalization 放在 `F(x)` 之前还是之后）训练稳定性差异很大。直觉上 pre-norm 让梯度路径更"干净"，但 post-norm 在某些任务上效果更好。这个 trade-off 我没想清，需要单独节点。
- 残差块的 `F(x)` 通常包含两层 conv / Linear，**为什么不是一层或三层**？这个工程选择有理论依据还是纯经验？
- 残差连接和 highway network 的 gating 机制本质上是同一思想（保留一条恒等路径），但 ResNet 没用 gate 也表现得很好。**gate 在残差里到底是不是必要**？这条线 Transformer 的 LayerNorm + residual 给出了一种回答（用 norm 替代 gate），但和 highway 的对比我没看到严谨论证。
