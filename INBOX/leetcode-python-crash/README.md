# LeetCode Python Crash

目标：在急投简历前，用 1-2 天恢复刷题手感，并把 Go 习惯迁移到 Python 刷题写法。

约束：

- 一年没写算法题，需要快速恢复题型判断能力。
- Python 语法不熟，容易把 Go 的写法和直觉带过来。
- 不默认已经理解 Go 的数据结构底层原理；每个主题先讲清楚底层机制，再做题。
- 只学 LeetCode / 面试题相关 Python，不做全面 Python 教程。

内容：

- `day1-2-python-crash-for-lc.ipynb`：4-6 小时的 Python 刷题 crash notebook。
- 每个主题先讲数据结构/机制底层原理，再给同一个小问题的 Python 和 Go 对照实现。
- Python 示例代码都包含显式输入、输出打印和 `assert` 校验，执行单元后可以直接看到结果。
- Go 只作为对照样例，帮助看清语言差异；练习只放 Python。
- 每个主题包含 2 个 Python 动手练习和参考实现。
- 最后包含 Python 异步 `asyncio` 与 Go goroutine/channel 直觉的对照讲解。
