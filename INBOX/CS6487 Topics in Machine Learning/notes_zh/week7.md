## 1. 预备知识：范数、不等式与二次型

### 1.1 内积与范数

原文要点：

- 使用标准内积 `⟨x, y⟩ := x^⊤ y`，以及欧几里得范数 `||x|| := sqrt(x^⊤ x)`。
- 回顾柯西-施瓦茨不等式：
  `|⟨x, y⟩| <= ||x|| ||y||`
- 回顾三角不等式：
  `||x + y|| <= ||x|| + ||y||`

注释：

- 这部分是在统一后续证明所需的几何语言。
- 后面所有关于下降、稳定性、收敛率的推导，都会反复调用这些最基础的范数不等式。

### 1.2 分析中常用的不等式

原文要点：

- Young 不等式（标量形式）。对任意 `a, b in R` 与 `eta > 0`，
  `ab <= (eta / 2) a^2 + (1 / 2eta) b^2`
- 配方法（向量形式）。对任意 `u, v in R^d` 与 `eta > 0`，
  `||u - v||^2 = ||u||^2 - 2⟨u, v⟩ + ||v||^2`
  因而
  `-2⟨u, v⟩ <= eta ||u||^2 + (1 / eta) ||v||^2`
- 讲义强调：这两个不等式本质上是同一类技巧，作用是把交叉项换成平方项。

注释：

- 证明里最常见的困难是出现 `⟨u, v⟩` 这类混合项。
- 这些不等式的作用就是把它们改写成可控、可求和的平方范数。

### 1.3 凸性与强凸性（快速回顾）

原文要点：

- 定义 1.1（凸性）。可微函数 `f` 若对任意 `x, y in R^d` 满足
  `f(y) >= f(x) + ⟨∇f(x), y - x⟩`
  则称为凸函数。
- 定义 1.2（`mu`-强凸）。若 `mu > 0` 且对任意 `x, y` 满足
  `f(y) >= f(x) + ⟨∇f(x), y - x⟩ + (mu / 2) ||y - x||^2`
  则称为 `mu`-强凸。
- 备注 1.1。强凸性意味着极小点唯一，并且在适当步长下，GD 具有几何收敛。现代 ML 中目标往往是非凸的，但这些结论对理解问题仍然非常有帮助，且对二次模型是精确成立的。

注释：

- 凸性提供一阶下界。
- 强凸性额外提供“向上托住”的二次曲率，这正是几何收敛出现的原因。

## 2. 泰勒定理与下降引理

### 2.1 带余项的泰勒展开

原文要点：

- 定理 2.1（多元二阶泰勒定理）。若 `f` 二次连续可微，则对任意点 `x` 和步长 `s`，
  `f(x + s) = f(x) + ⟨∇f(x), s⟩ + (1 / 2) s^⊤ ∇^2 f(x + tau s) s`
  其中 `tau in (0, 1)`。
- 讲义说明：若能沿线段 `x + tau s` 对 Hessian 做上界，就能得到 `f(x + s)` 的可用上界。这是把局部曲率与全局算法进展连接起来的技术桥梁。

注释：

- 泰勒展开告诉我们：一阶模型之所以不够，是因为还差一个由 Hessian 控制的二阶余项。
- 接下来引入的光滑性，就是用来统一控制这个余项的。

### 2.2 L-光滑性与下降引理

原文要点：

- 定义 2.1（L-光滑 / 梯度 Lipschitz）。
  若对所有 `x, y` 有
  `||∇f(x) - ∇f(y)|| <= L ||x - y||`
  则称 `f` 为 `L`-光滑。
- 引理 2.1（下降引理）。若 `f` 可微且在包含 `x, y` 线段的凸集上 `L`-光滑，则
  `f(y) <= f(x) + ⟨∇f(x), y - x⟩ + (L / 2) ||y - x||^2`
- 证明思路：
  令 `phi(t) = f(x + t(y - x))`，
  把 `phi(1) - phi(0)` 写成积分，
  再加减 `⟨∇f(x), y - x⟩`，
  最后用柯西-施瓦茨和 `L`-光滑性。
- 备注 2.1。下降引理是一阶方法分析中使用最频繁的不等式，它给出了一个二次上界模型。

注释：

- 这是整讲最核心的结论。
- 有了它，就能把未知的真实目标函数替换成一个可以严格控制的局部二次上界。

### 2.3 等价刻画

原文要点：

- 若 `f` 二次可微，则 `L`-光滑等价于 Hessian 满足统一上界
  `∇^2 f(x) ⪯ L I`

注释：

- 这给出光滑性的曲率解释：任意方向上的向上弯曲都不会超过 `L`。

## 3. 光滑性的推论

### 3.1 单步下降不等式

原文要点：

- 令 `y = x - alpha ∇f(x)` 代入下降引理，可得
  `f(x - alpha ∇f(x)) <= f(x) - alpha (1 - L alpha / 2) ||∇f(x)||^2`
- 因此任意
  `alpha in (0, 2 / L)`
  都能保证下降。

注释：

- 这是第一个严格的步长条件。
- 它解释了为什么 `alpha <= 1 / L` 会成为最常见、最稳妥的理论步长。

### 3.2 光滑且凸时，梯度更“规整”

原文要点：

- 引理 3.1（梯度的共胁迫性）。若 `f` 既凸又 `L`-光滑，则对任意 `x, y`，
  `⟨∇f(x) - ∇f(y), x - y⟩ >= (1 / L) ||∇f(x) - ∇f(y)||^2`
- 备注 3.1。该引理在近端方法、算子分裂、单调算子分析中都非常核心。

注释：

- 这比单纯的 Lipschitz 连续更强。
- 也正因为凸 + 光滑如此“规整”，很多高级优化理论都建立在这个组合上。

## 4. 收敛与速率的含义

### 4.1 优化中的“收敛”是什么意思

原文要点：

- 函数值收敛：`f(x_k) -> f*`
- 迭代点收敛：`x_k -> x*`
- 非凸情形下的一阶平稳性：`||∇f(x_k)|| -> 0` 或 `min_{0 <= t < k} ||∇f(x_t)|| -> 0`
- 讲义指出：在非凸 ML 目标里，GD 的标准确定性保证通常只是收敛到一阶平稳点。

注释：

- 在正式证明速率前，讲义先澄清目标是什么。
- 对非凸问题，最基础的目标不是全局最优，而是梯度足够小。

### 4.2 速率：次线性与线性

原文要点：

- 定义 4.1。设 `(a_k)` 是趋于 `0` 的非负序列。
- 次线性速率：`a_k = O(1/k)` 或 `O(1/sqrt(k))`
- 线性（几何）速率：`a_k <= C rho^k`，其中 `rho in (0, 1)`
- 备注 4.1。在 ML 中，“线性收敛”指几何衰减，而不是画成直线。

注释：

- 这套术语会在后面比较非凸、凸、强凸三种情形时反复用到。

## 5. 梯度下降：算法与收敛性

### 5.1 算法

原文要点：

- 常步长梯度下降：
  `x_(k+1) = x_k - alpha ∇f(x_k)`
- 讲义把它解释为反复最小化局部二次模型
  `m_k(y) := f(x_k) + ⟨∇f(x_k), y - x_k⟩ + (1 / 2alpha) ||y - x_k||^2`
  该模型的极小点正好是 `x_k - alpha ∇f(x_k)`。

注释：

- 这说明 GD 不是随意沿负梯度走一步，而是每一步都在解一个局部代理问题。

### 5.2 非凸保证：梯度范数变小

原文要点：

- 假设 `f` 是 `L`-光滑的，且下界为 `f* > -infinity`。
- 若 `alpha in (0, 1 / L]`，则
  `f(x_(k+1)) <= f(x_k) - (alpha / 2) ||∇f(x_k)||^2`
- 对 `k = 0` 到 `T - 1` 求和，得到
  `min_{0 <= k <= T-1} ||∇f(x_k)||^2 <= 2(f(x_0) - f*) / (alpha T)`
- 命题 5.1。取 `alpha = 1 / L`，则
  `min_{0 <= k <= T-1} ||∇f(x_k)|| <= sqrt(2L(f(x_0) - f*) / T)`
- 因而要使 `min ||∇f(x_k)|| <= epsilon`，只需
  `T = O(L(f(x_0) - f*) / epsilon^2)`

注释：

- 这是光滑非凸优化中最经典的确定性复杂度结果。
- 保证作用于前 `T` 步中的“最好一步”，而不一定是最后一步。

### 5.3 凸情形：函数值 `O(1/k)`

原文要点：

- 现在再加上 `f` 凸、`L`-光滑且极小点 `x*` 存在。
- 当 `alpha = 1 / L` 时，可证明
  `f(x_k) - f* <= (L / 2k) ||x_0 - x*||^2`
- 命题 5.2。若 `f` 凸且 `L`-光滑，GD 使用 `alpha = 1 / L`，则
  `f(x_k) - f* = O(1/k)`

注释：

- 与非凸情形相比，凸性把结果从“梯度小”升级成“目标函数值按 `O(1/k)` 衰减”。

### 5.4 条件数备注

原文要点：

- 备注 5.2。定义条件数 `kappa := L / mu`。
- 则 `(1 - mu / L)^k ≈ exp(-k / kappa)`。
- 当 `kappa` 很大时，GD 会显著变慢。

注释：

- 这正是后面引入动量与预条件的动机。

## 6. 实践中的步长选择

### 6.1 基于光滑性的常数步长

原文要点：

- 若已知 `f` 是 `L`-光滑，则最稳妥的选择是
  `alpha = 1 / L`
- 若还知道 `mu` 且函数强凸，则更好的调参常为
  `alpha = 2 / (L + mu)`

注释：

- `1 / L` 是最标准的理论步长。
- `2 / (L + mu)` 更偏向于二次 / 强凸模型下的最优收缩。

### 6.2 回溯线搜索

原文要点：

- 若 `L` 不知道，可从一个试探步长开始，不断乘 `beta in (0, 1)` 缩小，直到满足
  `f(x - alpha ∇f(x)) <= f(x) - (alpha / 2) ||∇f(x)||^2`
- 讲义指出：这样得到的步长在局部上会像 `1 / L`。

注释：

- 课程也明确提到：深度学习里由于 mini-batch 噪声，精确线搜索并不常见。

### 6.3 递减步长

原文要点：

- 对确定性光滑优化，常数步长通常最好。
- 递减步长 `alpha_k ↓ 0` 更适合随机梯度方法。

注释：

- 这段在强调：确定性 GD 理论与随机优化实践应分开看。

## 7. 动量：为什么有效，以及如何分析

### 7.1 动机：震荡与病态条件数

原文要点：

- 考虑二次目标 `f(x) = (1/2) x^⊤ H x`，其特征值位于 `[mu, L]`。
- 在特征坐标下，GD 退化为
  `z_(k+1) = (1 - alpha lambda) z_k`
- 稳定性要求 `alpha in (0, 2 / L)`。
- 当 `kappa = L / mu` 很大时，沿小曲率方向前进会极慢，且轨迹会出现 zig-zag。

注释：

- 这是理解病态优化最干净的二次模型。

### 7.2 Heavy-ball（Polyak）动量

原文要点：

- 定义 7.1。给定步长 `alpha > 0` 和动量系数 `beta in [0, 1)`，
  `x_(k+1) = x_k - alpha ∇f(x_k) + beta (x_k - x_(k-1))`
- 等价速度形式：
  `v_(k+1) = beta v_k + ∇f(x_k)`，
  `x_(k+1) = x_k - alpha v_(k+1)`
- 备注 7.1：
  若梯度长期方向一致，动量会累积它们；
  若梯度来回震荡，动量会像低通滤波器一样抑制高频变号。

注释：

- 这两条直觉正是工程上解释动量最常用的说法。

### 7.3 二次模型下的稳定性图像

原文要点：

- 在特征方向 `lambda` 上，heavy-ball 递推变成
  `z_(k+1) = (1 - alpha lambda + beta) z_k - beta z_(k-1)`
- 讲义解释：
  在陡峭方向上，反馈项有助于抵消过冲；
  在平坦方向上，由于允许使用更大的 `alpha`，收敛会更快。
- 命题 7.1。对特征值在 `[m, L]` 的二次问题，存在参数 `(alpha, beta)` 使
  `||e_k||_2 <= C ((sqrt(kappa) - 1) / (sqrt(kappa) + 1))^k ||e_0||_2`

注释：

- 这给出 heavy-ball 的核心收益：条件数依赖可从 `kappa` 级改善到 `sqrt(kappa)` 级。

### 7.4 Nesterov 动量（前瞻）

原文要点：

- Nesterov 加速梯度写作
  `y_k = x_k + beta (x_k - x_(k-1))`
  `x_(k+1) = y_k - alpha ∇f(y_k)`
- 与 heavy-ball 的区别是：梯度在前瞻点 `y_k` 处计算。
- 定理 7.1（非正式）。对凸且 `L`-光滑的目标，Nesterov 类方法可达到
  `f(x_k) - f* = O(1 / k^2)`

注释：

- 讲义说明这一定理的完整证明更复杂，通常需要 estimate sequence 或 Lyapunov 函数。
- 在实际深度学习框架中，很多“momentum”实现更接近 heavy-ball。

### 7.5 参数经验法则

原文要点：

- 在已知 `mu, L` 的确定性二次模型中，可选
  `alpha = 4 / (sqrt(L) + sqrt(mu))^2`
  `beta = ((sqrt(L) - sqrt(mu)) / (sqrt(L) + sqrt(mu)))^2`
- 在深度学习实践中，常见默认值为 `beta in {0.9, 0.95, 0.99}`，再调学习率。
- 备注 7.3。对 SGD 而言，动量既可能帮助平滑噪声、加速优化，也可能放大偏差或导致过冲。

注释：

- 课程在这里明确区分了“精确二次理论”与“随机非凸实践”。

## 8. 总结与记忆点

原文要点：

- `L`-光滑性推出下降引理，下降引理给出 GD 的严格下降界。
- 当 `alpha <= 1 / L` 时，GD 是下降方法，并具有：
  `非凸：min_{t < T} ||∇f(x_t)|| = O(1 / sqrt(T))`
  `凸：f(x_T) - f* = O(1 / T)`
  `强凸：几何收敛，因子约为 1 - 1 / kappa`
- 步长是稳定性的主旋钮。
- 动量通过引入惯性，缓解 zig-zag 并改善条件数影响。

注释：

- 第七讲给后两讲打下基础：
  先理解一阶方法为何收敛，
  再理解预条件和景观为何重要。
