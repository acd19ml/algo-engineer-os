## 1. 多变量优化中的临界点

### 1.1 一阶与二阶视角

原文要点：

- 设 `f : R^d -> R` 二次可微。
- 若某点 `x_hat` 满足
  `∇f(x_hat) = 0`
  则称为临界点。
- 一元函数里，我们看 `f''(x)` 的符号；多元情形里，对应对象是 Hessian 矩阵 `∇^2 f(x_hat)`。
- 定义 1.1（局部最小 / 最大 / 鞍点）：
  若 `f(x_hat) <= f(x)` 对附近所有 `x` 成立，则是局部最小；
  若 `f(x_hat) >= f(x)`，则是局部最大；
  否则是鞍点。
- 临界点附近的二阶泰勒展开为
  `f(x_hat + h) = f(x_hat) + (1 / 2) h^⊤ ∇^2 f(x_hat) h + o(||h||^2)`

注释：

- 因此，临界点附近的局部几何完全由 Hessian 的二次型控制。

### 1.2 Hessian 判别法

原文要点：

- 命题 1.1（多变量 Hessian 判别法）：
  若 `x_hat` 是 `C^2` 函数的临界点，则
  `∇^2 f(x_hat) ≻ 0` 时是严格局部极小值；
  `∇^2 f(x_hat) ≺ 0` 时是严格局部极大值；
  若同时存在正负特征值，则是严格鞍点；
  若仅为半正定 / 半负定，则二阶信息不足以下结论。
- 讲义备注：
  正特征值表示向上弯曲，负特征值表示向下弯曲，正负混合意味着鞍形。

注释：

- `lambda_min(∇^2 f)` 很关键，因为它为负就意味着存在负曲率方向。

### 1.3 为什么严格鞍点在算法上较“友好”

原文要点：

- 若一个鞍点的 Hessian 至少有一个严格负特征值，则称其为严格鞍点。
- 这种负曲率方向提供了局部逃逸通道。
- 定义 1.2（严格鞍点性质）：
  若一个函数的每个临界点，要么是局部极小值，要么满足
  `lambda_min(∇^2 f) < 0`
  则称该函数具有严格鞍点性质。

注释：

- 这就是课程想强调的“良性非凸景观”定义：
  坏点可以有，但它们必须是不稳定、可逃离的。

## 2. 矩阵感知：一个干净的景观模型

### 2.1 矩阵感知问题

原文要点：

- 设 `M* in R^(d1 x d2)` 是未知低秩矩阵，`rank(M*) = r`。
- 观测线性测量
  `y_i = ⟨A_i, M*⟩`，`i = 1, ..., m`
- 感知算子定义为
  `[A(M)]_i = ⟨A_i, M⟩`
- 无噪声问题可写成：
  找到 `M`，使得 `A(M) = A(M*)` 且 `rank(M) <= r`
- 常见的非凸分解写法是
  `M = U V^⊤`
  对应目标
  `min_(U,V) (1 / 2m) sum_i ⟨A_i, U V^⊤ - M*⟩^2`
- 若目标是对称 PSD，则常用更简单的分解
  `M = X X^⊤`

注释：

- 非凸性来自因子化参数化，而不是来自测量模型本身。

### 2.2 与二次激活两层网络的形式联系

原文要点：

- 考虑带二次激活的一隐藏层网络：
  `y(x; U) = sum_j (u_j^⊤ x)^2 = x^⊤ U U^⊤ x = ⟨x x^⊤, U U^⊤⟩`
- 因此网络参数只通过 PSD 矩阵
  `M = U U^⊤`
  起作用。
- 若数据由教师矩阵 `M* = U* U*^⊤` 生成，则经验平方损失就是
  `min_U (1 / 2m) sum_i ⟨x_i x_i^⊤, U U^⊤ - M*⟩^2`
- 也就是说，二次激活两层网络是对称矩阵感知的一个特例，感知矩阵为 `A_i = x_i x_i^⊤`。

注释：

- 这就是为什么矩阵感知并不是脱离 ML 的纯数学例子，它本身已经对应了一个简单的神经网络训练问题。

## 3. RIP 与“数据为什么会改善几何”

### 3.1 限制等距性质 RIP

原文要点：

- 定义 3.1。若感知算子 `A` 对所有秩不超过 `r` 的矩阵 `M` 满足
  `(1 - delta_r) ||M||_F^2 <= (1 / m) ||A(M)||_2^2 <= (1 + delta_r) ||M||_F^2`
  则称其满足 `(r, delta_r)`-RIP。
- 讲义解释：这表示 `A` 在低秩集合上近似保持 Frobenius 范数。

注释：

- RIP 的本质是：低秩结构在测量之后没有被严重扭曲。

### 3.2 向量版本的简化理解：高斯感知与 RIP

原文要点：

- 设 `Phi in R^(m x N)` 的元素独立服从 `N(0, 1/m)`。
- 对固定向量 `x`，
  `E ||Phi x||_2^2 = ||x||_2^2`
- 还满足浓缩不等式
  `Pr(| ||Phi x||_2^2 - ||x||_2^2 | >= eps ||x||_2^2) <= 2 exp(-m c_0(eps))`
- 若想让该结论对所有 `k`-稀疏向量同时成立，常见证明分三步：
  固定一个支撑集；
  对单位球做 epsilon-net 覆盖；
  再对所有支撑集做并联合并。

注释：

- 课程强调的不是常数，而是机制：
  各向同性 + 浓缩 + 覆盖 => 在结构化集合上近似等距。

### 3.3 回到矩阵感知

原文要点：

- 同样的逻辑也适用于低秩矩阵。
- 对高斯感知矩阵 `A_i`，有
  `E ⟨A_i, M⟩^2 = ||M||_F^2`
- 再加上浓缩和对低秩流形的 epsilon-net，即可得到 RIP。
- 讲义给出的重要启发是：
  更多样本 `m`
  => 更强浓缩
  => 更小的 `delta_r`
- 并特别指出：数据增加不仅降低统计误差，也会改善经验目标的几何形状。

注释：

- 这是整讲很重要的一句：数据量不仅影响“估计准不准”，也影响“优化好不好做”。

## 4. 一个可完全分析的 rank-1 景观示例

原文要点：

- 研究玩具问题
  `min_x (1 / 2) ||x x^⊤ - M*||_F^2`
  其中 `M* ⪰ 0`，且 `A` 就是恒等映射。
- 这对应最理想的矩阵感知情形：没有测量畸变，也就是 `delta_(2r) = 0`。

### 4.1 临界点的显式求解

原文要点：

- 梯度为
  `∇f(x) = 2(x x^⊤ - M*)x = 2(||x||^2 x - M* x)`
- 因而 `∇f(x) = 0` 当且仅当
  `M* x = ||x||^2 x`
- 所有临界点因此要么是 `x = 0`，要么是 `M*` 的特征向量按对应特征值平方根缩放：
  `x_hat in {0, ±sqrt(lambda_1) u_1, ..., ±sqrt(lambda_n) u_n}`

注释：

- 这个模型可贵的地方在于：临界点集合可以完全写出来。

### 4.2 Hessian 分类

原文要点：

- Hessian 为
  `∇^2 f(x) = 4x x^⊤ + 2||x||^2 I - 2M*`
- 在非零临界点 `x = ±sqrt(lambda_i) u_i` 处，
  `∇^2 f(±sqrt(lambda_i) u_i) = 4 lambda_i u_i u_i^⊤ + sum_(j != i) 2(lambda_i - lambda_j) u_j u_j^⊤`
- 因而：
  `±sqrt(lambda_1) u_1` 是二阶局部极小值；
  `±sqrt(lambda_i) u_i`（`i >= 2`）是严格鞍点；
  `x = 0` 在 `M* ≻ 0` 时是局部极大值，更一般情形下是退化鞍点 / 局部极大值。
- 讲义结论：
  所有二阶局部极小值都是全局解，其他临界点要么是严格鞍点，要么是局部极大值。

注释：

- 这就是课程想要展示的“良性非凸景观”的最干净例子。

## 5. 为什么小 RIP 能排除伪局部极小值

### 5.1 对称目标与理想目标

原文要点：

- 考虑对称因子化目标
  `f(X) := (1 / 2m) ||A(X X^⊤ - M*)||_2^2`
- 若 `A` 在秩 `2r` 的矩阵上就是恒等映射，则它退化为
  `(1 / 2) ||X X^⊤ - M*||_F^2`
- 讲义将核心思想概括为：
  小 RIP
  => 经验二次型在低秩方向上接近 Frobenius 范数
  => 真实景观只是理想景观的小扰动

注释：

- 这几乎就是整个景观定理的一句话版本。

### 5.2 非正式景观定理

原文要点：

- 定理 5.1（矩阵感知的非正式景观定理）。
  若感知算子 `A` 满足足够小常数下的 `(2r, delta)`-RIP，则：
  每个局部极小值都对应真实矩阵 `M*`，
  每个其他临界点都存在严格负曲率方向。

注释：

- 也就是说：
  没有伪局部极小值；
  所有非全局临界点都是严格鞍点。

### 5.3 证明思路：梯度与 Hessian 的分解

原文要点：

- 令 `E = X X^⊤ - M*`。
- 梯度可写成
  `∇f(X) = [A* A(E) / m] X`
- 沿方向 `Δ` 的 Hessian 二次型为
  `∇^2 f(X)[Δ, Δ] = (1 / m) ||A(XΔ^⊤ + ΔX^⊤)||_2^2 + (2 / m) ⟨A* A(E), ΔΔ^⊤⟩`
- 第一项总是非负。
- 第二项可能为负。
- 要证明严格鞍点，就要找某个方向 `Δ`，让第二项压过第一项。

注释：

- 这就是技术核心：
  一个永远非负的“变化项”，
  加上一个可能带来负曲率的“误差项”。

### 5.4 与真值因子的最佳对齐

原文要点：

- 在小 RIP 下，可以把 `A* A / m` 近似看成单位映射。
- 设 `X* X*^⊤ = M*`，并令 `R` 为最佳正交对齐矩阵。
- 定义
  `Δ(X) = X - X* R`
- 则有
  `E = X X^⊤ - X* X*^⊤ = XΔ^⊤ + ΔX^⊤ - ΔΔ^⊤`
  从而
  `XΔ^⊤ + ΔX^⊤ = E + ΔΔ^⊤`

注释：

- 之所以要引入 `R`，是因为因子分解本身有旋转不唯一性。

### 5.5 一阶临界条件导致关键消去

原文要点：

- 对一阶临界点 `X_hat`，在近似 `A* A / m ≈ I` 下有
  `∇f(X_hat) = E X_hat = 0`
- 把它与 `Δ` 做内积，可推出
  `⟨E, X_hatΔ^⊤ + ΔX_hat^⊤⟩ = 0`
- 再用
  `X_hatΔ^⊤ + ΔX_hat^⊤ = E + ΔΔ^⊤`
  得到
  `0 = ||E||_F^2 + ⟨E, ΔΔ^⊤⟩`
  因而
  `⟨E, ΔΔ^⊤⟩ = -||E||_F^2`

注释：

- 这是讲义中的关键消去步骤：把一阶临界性直接转化成一个负曲率来源。

### 5.6 Hessian 最终为何为负

原文要点：

- 代回后，
  `∇^2 f(X_hat)[Δ, Δ] ≈ ||E + ΔΔ^⊤||_F^2 - 2||E||_F^2`
  `= ||ΔΔ^⊤||_F^2 - 3||E||_F^2`
- 所以它变成
  正项 `||ΔΔ^⊤||_F^2`
  与负项 `-3||E||_F^2`
  的竞争。
- 若 `Δ` 很小，则 `E` 对 `Δ` 近似是一阶，而 `ΔΔ^⊤` 是二阶，因此负项主导。
- 若 `Δ` 很大，则 `ΔΔ^⊤` 主导两边，但表达式仍然为负。
- 因此只要 `Δ != 0`，就有 `∇^2 f(X_hat)[Δ, Δ] < 0`。

注释：

- 这一步最终说明：只要不是全局解，就一定存在逃逸方向。

## 6. 这对神经网络意味着什么

原文要点：

- 矩阵感知比现代深度网络简单得多，但它传达的教训很重要：
  非凸性本身不是敌人。
- 若数据模型与参数化方式诱导出合适的几何，那么非凸目标依然可以拥有全局友好的景观。
- 讲义总结出的反复出现的模式是：
  对称性带来多个等价全局极小值；
  坏临界点往往变成严格鞍点；
  好的数据假设让经验目标接近一个行为良好的总体目标。

注释：

- 这就是本讲从矩阵感知回到现代机器学习的落点。
