# Target Roles

每个目标岗位一个 `.md`。当前阶段必填两个：

- `summer-intern-agent-engineer.md` — 暑期实习目标态（半个月内开始投递）
- `newgrad-agent-engineer.md` — 秋招目标态（9 月）

## 单个岗位页该有什么

```markdown
# {Role Title}

## 目标公司 / 类型
- 一线大厂 agent / multi-agent 团队
- AI startup（具体方向）
- ...

## 核心能力要求（从 JD / 内推 / 学长经验提炼）
- 必须会：...
- 优先会：...
- 加分项：...

## 典型面笔试问题
- 技术：...
- 系统设计：...
- 行为：...

## 我目前的对照
- 已具备：链 cv.md bullet 或 KNOWLEDGE/x
- 缺口：链 skill-gap.md 对应行
- 优势项（同竞争者比）：...

## 参考来源
- JD 原文（链接 / 截图）
- 学长 / 内推 / 群信息
- 招聘平台
```

## 怎么长出来

两种方式都行：
1. **手动**：你自己研究 JD / 找内推问 / 看招聘平台，写进来
2. **对话驱动**：和 LLM 对话讨论某个岗位，把对话存进 `INBOX/career/dialogue_logs/`，triage 时 LLM 会建出对应的 `target-roles/x.md`
