## JD 快照

华为集团 IT（质量与流程 IT 部）推动公司自身数智化转型，业务应用与 IT 装备服务部负责打造企业级大规模 SaaS 产品，包括 MetaERP、MetaCRM、MetaIT、MetaSCM、Welink 等，支撑研发、财经、营销销售、服务交付、供应链、采购、生产制造、人力资源、行政办公等全业务场景。

## 匹配判断

### 强匹配

- **企业级 Agent 落地**：七牛云 AI 运维 Agent 覆盖多源运维数据、MCP 工具封装、多智能体 RCA 和主动巡检，和华为企业级 SaaS 智能化转型语境接近。
- **LangChain / LangGraph 多智能体**：Neo DeepResearch 服务和七牛云 RCA Agent 都能支撑多节点编排、任务规划、工具调用和报告生成。
- **RAG / 工具路由**：Neo ReAct 区块链问答 Agent 的双路 RAG 路由、62 工具语义索引和 Top10 → Top5 候选上下文注入，可对应 Code Agent 场景下的工具选择和上下文预算问题。
- **Harness Engineering**：已有 `PROBLEMS/agent-harness-boundary-map/`，可从 Context / Evaluation / Tool Repair / Memory / Boundary Probing 五类 Harness 讲工程控制层。
- **企业级 Memory / Skills**：Agent Memory 自主研究 + Neo RAG 工具路由可对应企业级 Memory、Skills 和 MCP 市场。
- **AI Native SaaS 认知**：七牛云项目里做过产品定位、MVP 收敛、API 定义、模块拆分和架构设计，能讲企业内部系统从传统流程向 AI 工作流演进。

### 风险点

- 华为集团 IT 可能更看重 Java / 后端工程 / 企业流程经验；简历表达需要突出工程交付，不只讲算法概念。
- 200+ AI 实习生招聘可能岗位池很宽，需沟通确认具体组别。
- 岗位含 AI 模型工程师、AI 应用工程师、AI 安全隐私工程师三类方向，后续需要确认具体组别和期望深度。

### 一句话定位

> 企业级多智能体 + RAG 工具路由 + AI Native SaaS 工程化经历，适合集团 IT 从企业应用到 Agent 化的转型岗位。

## 面试准备

| 可能追问 | 准备材料 |
|---|---|
| 你理解的 AI Native SaaS 是什么？ | 从元数据多租 / 流程驱动 → Agent 编排 / 工具调用 / 上下文工程 / 可观测评测回答 |
| Code Agent 和普通业务 Agent 的差异？ | Code Agent 的工具空间是 repo / issue / CI / test / PR，核心是可执行反馈闭环；业务 Agent 更强调业务数据与流程 |
| 七牛云项目怎么体现企业级？ | 多源数据、角色分工、MCP 工具封装、主动巡检、灰度场景动态阈值 |
| Neo RAG 路由为什么和 Code Agent 有关？ | 60+ 工具选择、上下文预算、工具 schema 描述、TopK 召回与精排，是 Code Agent 工具选择的同构问题 |
| 你怎么评估 Agent 是否有效？ | 结合 `PROBLEMS/agent-harness-boundary-map/` 讲 evaluation harness、轨迹指标、工具调用正确率和失败 case |
| 你理解的 Harness Engineering 是什么？ | 不是单个 prompt 技巧，而是围绕模型边界建立 Context / Evaluation / Tool Repair / Memory / Boundary Probing 控制层 |
| 企业级 Memory 和个人 Agent Memory 有什么不同？ | 企业级更强调权限、安全、审计、组织知识、跨领域协同；个人 memory 更偏偏好和项目上下文 |
